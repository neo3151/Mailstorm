# Prompt Maximizer Workflow

The user has provided a basic command or idea alongside this workflow. Your sole goal is to transform their simple instruction into a comprehensive, highly detailed prompt engineered to extract the **maximum potential** from an AI model.

## Phase 1: Analyze the Implicit Goal
1. Read the user's basic command carefully.
2. Determine the core objective, target audience, and ideal format of the task.
3. Identify exactly what context, rules, or constraints are missing but crucial for the best possible AI output.

## Phase 2: Construct the Maximized Prompt
Draft a massive, highly structured prompt. It should contain the following elements to guarantee maximum performance:
- **Role / Persona:** Define exactly *who* the AI should act as (e.g., "Act as a world-class senior architectural engineer..." or "You are an elite, award-winning copywriter...").
- **Context / Background:** Expand the missing context around the basic idea so the AI understands the "why."
- **Specific Instructions:** Break down the task into clear, highly specific, sequential steps.
- **Constraints & Rules:** Detail a strict list of things the AI *must* do and *must not* do (e.g., "Do not use passive voice," "Never use generic placeholder text").
- **Output Format:** Specify exactly how the output should be structured (e.g., "Use markdown headers, produce a table, use JSON format").
- **Thinking Process (Optional):** Instruct the AI to explicitly use step-by-step reasoning or a `<thinking>` block before producing its final answer.

## Phase 3: Deliver
1. **Do not execute the prompt yourself.** Your job is strictly to *write* the masterpiece prompt for the user to copy.
2. Present the fully maximized prompt to the user inside a single markdown text block so they can easily copy and paste it.
3. Provide a brief explanation of *why* you structured the prompt this way and what specific elements you added to unlock its maximum potential.
